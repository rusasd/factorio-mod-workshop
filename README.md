# ⚙ Мастерская модов Factorio / Factorio Mod Workshop

**Создавайте моды для Factorio без программирования — прямо в браузере.**
Make Factorio mods with zero coding — right in your browser.

🔗 **Сайт / Live site:** включите GitHub Pages в настройках репозитория (Settings → Pages)

## Возможности

- 🧱 **Новое содержимое**: предметы, сундуки, рецепты, общие настройки игры
  (скорость бега, инвентарь, размер стопок) — через простые формы.
- 👾 **Правка всего, что есть в игре**: враги, турели, машины, печи — загрузите
  каталог своей игры, и меняйте любые свойства в наглядном дереве. Учитываются
  все ваши установленные моды.
- 🧬 **Клоны**: новый враг или турель на основе существующих — с вашими характеристиками.
- 🖼 **Свои спрайты**: PNG-картинки и иконки попадают прямо в zip мода.
- 📦 **На выходе — готовый zip**: кладёте в папку модов игры, и всё работает.
- 🌐 Поддержка Factorio **1.1** и **2.x**, интерфейс на русском, переводы модов ru+en.
- 🔒 **Приватность**: сайт статический, всё происходит в вашем браузере,
  никакие данные никуда не отправляются.

## Как получить каталог своей игры (для вкладки «Изменить игру»)

- **Factorio 2.x / 1.1.107+**: запустите игру с параметром `--dump-data`,
  затем загрузите на сайт файл `%appdata%\Factorio\data-raw-dump.json`.
- **Старые версии**: скачайте на сайте мод-дампер, положите в папку модов,
  запустите игру до меню, выйдите — и загрузите `%appdata%\Factorio\factorio-current.log`.

## Технически

Один файл `index.html` без зависимостей: генерация Lua-кода, локализаций и
zip-архива (собственный ZIP-writer) — на чистом JavaScript. Проекты хранятся
в localStorage, каталог игры — в IndexedDB.
